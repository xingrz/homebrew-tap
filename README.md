# xingrz/homebrew-tap

[Homebrew](https://brew.sh) tap for [@xingrz](https://github.com/xingrz)'s projects.

## Usage

```sh
brew tap xingrz/tap
```

Then install any formula or cask it provides:

```sh
brew install xingrz/tap/<name>          # command-line tools
brew install --cask xingrz/tap/<name>   # apps
```

Browse what's available with `brew search xingrz/tap`. Once tapped, you can also
drop the `xingrz/tap/` prefix (e.g. `brew install <name>`).
